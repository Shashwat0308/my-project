# lab-mst (React Lazy Loading + Form Validation)

This folder contains a small Vite React demo that shows:

- ✅ **Component lazy loading** using `React.lazy()` + `Suspense`
- ✅ **Form validation** with inline errors and submission state

## How to run

1. Open a terminal in this folder:

   ```powershell
   cd lab-mst\Lazy-form
   npm install
   npm run dev
   ```

2. Open the link shown (usually `http://localhost:5173`) in your browser.
3. Click **Show form** to lazy-load the form component.
4. Fill in the form to see errors and submit.
